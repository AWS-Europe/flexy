# Flexy
Flexy is minimal CSS framework made with Flex

* Minimal. It is only 0.33 KB minified + gzip. 
* Responsive. 
* No unnecessary nesting.
* Fluid Column
* Flexible the main width can be any number or uint ex: 960px, 90% or whatever you like.
